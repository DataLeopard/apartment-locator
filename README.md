# Apartment Locator — Georgetown, TX

Apartment locator dashboard for real estate agents working the Georgetown, TX market. Search, filter, and manage apartment listings alongside active client leads — with commission calculations, property details, and lead-to-listing matching built in.

## Features

- **Property database** — 6 Georgetown apartment communities with rent ranges, square footage, deposit info, amenities, and unit availability
- **Smart filters** — filter by bedroom count, max rent, pet policy, and listing status (active, low inventory, waitlist)
- **Commission tracker** — each listing shows commission percentage with a quick calculator for per-placement earnings
- **Agent notes** — personal notes on each property (move-in specials, approval speed, best-fit client types)
- **Contact directory** — property manager names and phone numbers on every listing
- **Lead management** — track active clients with budget, bedroom needs, pet requirements, and move-in dates
- **Lead matching** — each lead shows which properties they've been matched to
- **Status pipeline** — leads flow through searching, touring, applied, and placed stages

## Getting Started

```bash

npm run build     # Build for production

```
